## 使用方式

1. 先看 00-checklist，了解整体练习计划
2. 按优先级逐个练习：自我介绍 → 管理短板 → PC 开播助手 → 3D 球场 → AIRead SDK
3. 练习到脱稿能自然说出为止（不是背稿，是内化成自己的语言）
4. 最后用 06-faq-bank 做随机问答练习

## 核心原则

- 从"我做了什么"升级为"我为什么这么做"
- 每个回答都有结构（三段式），不是散乱的
- 管理短板不回避，用对等经验补位
- 展示技术判断力，不只是技术执行力
