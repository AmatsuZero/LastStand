# 自我介绍（2分钟版）

## 定位句（10秒）

我是 iOS 客户端工程师，在腾讯体育有 3 个全链路项目 Owner 经验，覆盖 3D 图形、跨平台架构和音视频推流。

## 核心项目快速过（各20秒，共60秒）

**3D 球场** — 非版权网球赛事的 3D 动画直播系统。从引擎选型（Cocos Creator）到数据驱动播放，再到服务端链路和客户端优化，我负责全链路。解决了"没版权也能做类直播体验"的业务痛点。

**PC 开播助手** — Electron + OBS Node 架构的桌面端直播工具。最难的工程问题是跨进程视频渲染，我设计了"挖洞+零拷贝GPU渲染"方案。2个月从0到上线。

**AIRead SDK** — 基于 KMM 的跨端语音播报 SDK。核心设计是把状态机统一在 commonMain，播放器下沉到平台层，避免双端状态漂移。Android/iOS 双端交付。

## 差异化收尾（30秒）

除了业务项目，我持续在写技术博客——17 篇 objc4 runtime 源码分析、16 章 AI Agent 算法拆解、以及团队的 AI 辅助开发工作流框架（OSC）。这些让我在技术深度、视野和工程化思维上都有系统积累，也是我面客户端负责人岗位的底气。
